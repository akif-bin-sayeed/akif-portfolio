Upload Guide

How to publish edits

  1. Make changes in this dashboard.
  2. Click Download profile.json.
  3. Go to your GitHub repository.
  4. Open data/profile.json.
  5. Click the pencil/edit button or upload/replace the file.
  6. Commit changes.
  7. Netlify redeploys automatically in 1–3 minutes.

Contact form activation

After deploying, submit one message from the live contact form. Then open Netlify → your project → Forms. The form should appear there.

Android workflow

Open this dashboard from Chrome on Android, edit text, choose a photo, download profile.json, then upload it to GitHub using the GitHub mobile app or browser.
